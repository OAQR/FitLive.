## Propósito y Alcance

Cubre la estructura del proyecto definida en <code class="artifact-id">pom.xml</code>, el árbol de dependencias completo, los plugins de compilación y los patrones de configuración específicos de IDE. ### Dependencias de Documentación de API

### Exclusión del Maven Wrapper

<p>El archivo <code class="artifact-id">.gitignore</code> incluye específicamente el JAR del Maven Wrapper mientras excluye otros artefactos de compilación:</p>
<CodeBlock language="text" code={`target/
!.mvn/wrapper/maven-wrapper.jar
!**/src/main/**/target/
!**/src/test/**/target/
`} />

<p>El prefijo <code class="artifact-id">!</code> indica una excepción a la regla de exclusión. Esto asegura que <code class="artifact-id">maven-wrapper.jar</code> sea commiteado al control de versiones, permitiendo a los desarrolladores compilar el proyecto sin necesidad de una instalación de Maven preexistente. El wrapper puede ser invocado usando <code class="artifact-id">./mvnw</code> en sistemas tipo Unix o <code class="artifact-id">mvnw.cmd</code> en Windows.</p>
