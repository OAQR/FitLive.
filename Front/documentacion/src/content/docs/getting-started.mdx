<p>Esta página proporciona instrucciones para compilar, ejecutar y desarrollar los microservicios     de FitLive localmente usando el Maven Wrapper. El sistema FitLive consiste en dos servicios Spring    Boot: <code class="artifact-id">servicio-clientes</code> (gestión de clientes) y <code      class="artifact-id">servicio-facturacion</code> (generación de facturas).</p>
<p>Para información sobre la arquitectura general del sistema y cómo interactúan los servicios,       consulte <a href="/system-architecture.mdx" class="content-link">Arquitectura del Sistema</a>. Para detalles de configuración y despliegue específicos de cada servicio, vea <a href="#"  class="content-link">Configuración y Despliegue de servicio-clientes</a> y <a href="#"     class="content-link">Configuración e Instalación de servicio-facturacion</a>.</p>

## Resumen del Maven Wrapper

<p>
    El Maven Wrapper (<code class="artifact-id">mvnw</code> y <code class="artifact-id">mvnw.cmd</code>) es un script de shell y un archivo batch que asegura que todos los desarrolladores usen la misma versión de Maven sin requerir una instalación local. Ambos servicios usan la versión 3.3.4 del Wrapper, que descarga Apache Maven 3.9.11.
</p>
