Esta página proporciona instrucciones para compilar, ejecutar y desarrollar los microservicios de FitLive localmente usando el Maven Wrapper. El sistema FitLive consiste en dos servicios Spring Boot: servicio-clientes (gestión de clientes) y servicio-facturacion (generación de facturas).

Para información sobre la arquitectura general del sistema y cómo interactúan los servicios, consulte Arquitectura del Sistema. Para detalles de configuración y despliegue específicos de cada servicio, vea Configuración y Despliegue de servicio-clientes y Configuración e Instalación de servicio-facturacion.

Prerrequisitos

Antes de compilar y ejecutar los servicios, asegúrese de que su entorno de desarrollo tenga lo siguiente instalado:

Requisito Versión Propósito
Java Development Kit (JDK) 22 Requerido para compilar y ejecutar aplicaciones Spring Boot 3.5.6.
Git Cualquier versión reciente Para clonar el repositorio y el control de versiones.
PostgreSQL (opcional) 12+ Base de datos de producción; por defecto, se utiliza la base de datos en memoria H2 para el desarrollo local.
IDE (opcional) IntelliJ IDEA, VS Code, STS, o NetBeans Para la edición de código y depuración (debugging).

Nota: La instalación de Maven no es requerida. El repositorio incluye scripts del Maven Wrapper que descargan y utilizan automáticamente la versión correcta de Maven (3.9.11) como se especifica en Back/servicio-clientes/.mvn/wrapper/maven-wrapper.properties 1-4

Fuentes: Back/servicio-clientes/.mvn/wrapper/maven-wrapper.properties diagramas de arquitectura del sistema

Resumen del Maven Wrapper

El Maven Wrapper (mvnw y mvnw.cmd) es un script de shell y un archivo batch que asegura que todos los desarrolladores usen la misma versión de Maven sin requerir una instalación local. Ambos servicios usan la versión 3.3.4 del Wrapper, que descarga Apache Maven 3.9.11.

Componentes del Maven Wrapper:

Archivo Plataforma Propósito
mvnw Unix/Linux/macOS Script de shell para sistemas compatibles con POSIX.
mvnw.cmd Windows Script de Batch/PowerShell para Windows.
.mvn/wrapper/maven-wrapper.properties Todas Configuración que especifica la versión de Maven y la URL de descarga.
.gitattributes Todas Asegura las terminaciones de línea correctas (LF para mvnw, CRLF para *.cmd).
