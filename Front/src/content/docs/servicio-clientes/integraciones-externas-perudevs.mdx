## Propósito y Alcance

<p>Este documento describe la integración entre <code class="artifact-id">servicio-clientes</code> y el servicio de API externa PeruDevs. PeruDevs proporciona validación y recuperación de información para documentos de identificación fiscal peruanos, específicamente RUC (Registro Único de Contribuyentes) y DNI (Documento Nacional de Identidad).</p>
<p>La integración permite al servicio de gestión de clientes recuperar y validar datos de identificación de empresas e individuos desde fuentes oficiales peruanas. Este documento cubre la implementación del adaptador, los endpoints de la API, los modelos de petición/respuesta y las estrategias de manejo de errores.</p>
<p>Para obtener información sobre cómo se utiliza esta integración dentro de la capa de servicio, consulte <a href="/docs/servicio-clientes/capa-servicio-logica-negocio" class="content-link">Capa de Servicio y Lógica de Negocio</a>. Para la exposición de los endpoints de la API, consulte <a href="/docs/servicio-clientes/endpoints-api-rest" class="content-link">Endpoints de la API REST</a>.</p>

## Estrategia de Manejo de Errores

<p>El <code class="artifact-id">PeruDevsAdapter</code> se inyecta en <code>ClienteServiceImpl</code>, donde se utiliza como un fallback cuando los datos del cliente no se encuentran en la base de datos local.
